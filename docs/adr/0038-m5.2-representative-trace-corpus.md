# ADR 0038: M5.2 Representative Ordered Expert-Trace Corpus

## Status

Accepted for M5.2-01 evidence capture.

## Context

The M5.1-00 authoritative trace contains one short Tier-A workload. It is
adequate for validating the first cache primitive, but it cannot establish
whether the observed 8 GiB behavior generalizes across languages, prompt
shapes, context lengths, or decode lengths.

## Decision

M5.2-01 records a deterministic corpus of individual ordered expert-request
traces. The corpus uses schema v2, with the v1 request fields retained and
fixture identity, workload metadata, generation metadata, KV-cache metadata,
and aggregate runtime counters added to the header. Each occurrence retains
the simulator/production fields: global ordinal, phase, decode step, absolute
position, layer, selected-expert rank, expert ID, key, payload bytes, hit,
load, and eviction count.

The primary capture configuration is F32, greedy seed 0, a one-expert
18,874,368-byte strict global-LRU cache, fixed KV capacity per fixture, and
explicit recording of every logical request. Canonical traces contain no
timestamps, process IDs, absolute paths, or timing values. Each trace is
captured twice and promoted only when the canonical bytes and SHA-256 match.

The frozen M5.1-00 Tier-A trace remains the control trace. Five existing
Tier-B fixtures are reused where suitable; the longer-context and longer-
decode cases are classified as Rust deterministic or trace-only fixtures
unless an independent cross-runtime reference exists.

## Consequences

The corpus supports independent trace validation and future selective replay.
The aggregate report is descriptive only: M5.2-01 does not simulate cache
policies and does not select a new RAM preset. The 8 GiB recommendation is
therefore classified as inconclusive until M5.2-02 performs the explicitly
deferred corpus simulation.

The M5.1-03 counter differences are preserved as a separate evidence record.
They are explained by modeled-overhead versus nominal-payload budget
accounting, so no runtime behavior change is required.
