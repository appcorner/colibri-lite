# M4.3 Phase Closure

M4.3 is closed as an evidence-driven evaluation phase. The authoritative
ordered Rust F32 path remains accepted. The first expert INT8 per-output-channel
format is rejected for full-model production because local accuracy did not
survive full-depth propagation, while group-128 remains promising but
unproven. Router INT8 is rejected and dense BF16 remains a future candidate
requiring compute and activation evidence.

The external Q4_K_M ik_llama result is a CPU performance reference only. It
demonstrates useful hardware capability and identifies memory residency,
optimized kernels, SIMD, threading, fused MoE, and graph reuse as material
architectural differences. It does not establish model-quality equivalence.

The project continues with a memory-hierarchy pivot. No quantized runtime or
optimization prototype is part of M4.3. The first next-phase experiment is a
trace-driven simulation of resident dense weights and expert-cache sizing at
1/2/4/8/16/24/32 binary GiB, followed by review before any runtime change.

M4.3 verdict: `completed_with_documented_quantization_rejection_and_optimization_pivot`.
