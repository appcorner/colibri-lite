five row intermediates and all 8,192 weighted outputs. The first PyTorch-versus-
ordered divergence is layer 0, token 0, at the sum-of-squares reduction. This
classifies the isolated RMSNorm difference as cross-runtime reduction-order
variance, not a Rust implementation defect.

No tolerance, runtime operation, router behavior, or later-layer path changed.
M4.2-02 remains incomplete pending review of the cross-runtime RMSNorm contract.
