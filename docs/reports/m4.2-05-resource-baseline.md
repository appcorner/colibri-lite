# M4.2-05 Resource and I/O Baseline

## Result

M4.2-05 passes. Three fresh-process runs preserved prompt IDs
`[9707, 11, 1879, 0]`, generated IDs `[1096, 374]`, both F32
`exact_match_safe` classifications, and the validated fixed-capacity KV-cache
behavior. No numerical, router, cache, artifact, or capacity contract changed.

The recorded host was Windows 11 Enterprise 10.0.26200 on an Intel Core
i7-1165G7 with 8 logical processors and 51,249,209,344 bytes of installed RAM.
Rust was 1.96.1 for `x86_64-pc-windows-msvc`.
