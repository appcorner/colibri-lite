# M4.2-02 Layer-1 Router Validation

## Result

The normal Rust F32 path passes from embedding through the complete Layer-0
streaming block and the Layer-1 router. Only the 27 experts selected by the
approved Layer-0 router were registered and loaded. No Layer-1 expert and no
Layer 2 operation executed.

Transformers F32 and Rust select identical Layer-1 experts for every token with
safe positive margins. All four primary classifications are
`exact_match_safe`. The independent BF16 classifications are all
`numerically_ambiguous`, although their observed IDs also match.

There is no mismatch and no isolated Layer-1 input diagnostic was run.
